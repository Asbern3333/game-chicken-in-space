# chicken-in-space from scratch

# description
- Making the game from scratch (without already built engine)
- Chicken-In-Space is a game that was my project for the course object-oriented class.
- It is about a plane in a space that shoots the moving chickens and tries to stay alive.
# installation
You can download it as a zip file form the green button called `code` on the main page of the game. Then Download Javac: https://adoptium.net/
Then:
Step 2 — Install the JDK (Correct Way)

Download JDK from:
https://adoptium.net/

Download:

Windows

x64

Latest LTS (Java 17 or 21)

Install it normally.

Step 3 — Add Java to PATH (This is the part most people skip)
After installation:
Press Windows Key
Search: Environment Variables
Click Edit the system environment variables
Click Environment Variables
Under System Variables, find Path
Click Edit
Click New
Add this path (example):
C:\Program Files\Eclipse Adoptium\jdk-21\bin 
(Adjust depending on where it installed.)
Click OK everywhere.
Step 4 — Restart PowerShell
Close VS Code.
Close PowerShell.
Open them again.
Now try:
javac -version
If it prints a version number, you’re good.
